HTML (Hypertext Markup Language)

  • Gives the page structure (Framework).
  • Elements users can see: Text, image, header, list.
  • Elements user can't see: Meta related, content division, semantic.
  • Structure:
    • Head: User cannot see this section. Helps browser render webpage.
    • Body: Seeable. Content shown to the user.

CSS (Cascading Style Sheets)

  • Visual design. Makes the page more readable.
  • 3 types: Inline, internal & external.
    • Inline: Uses a style attribute in a single line.
    • Internal: Set of rules in "Head" section of HTML file.
    • External: Uses a diferent file, linked to HTML.

Git (Version Control)

  • Version Control.
  • Cloud and Local repositories.
  • Uses "branches" to avoid affecting the main code.
  • You can test features.
  • Git workflow:
    • git add -A
    • git commit -m "Add comment".
    • git pull origin main.
    • git push origin "branch to upload".

JavaScript

  • Utilizes variables, loops and functions.
  • Variables can be stored in "Arrays", which can contain multiple types of data.
  • Linked to HTML files.
  • Control flow is the order in which a computer executes code in a script.
